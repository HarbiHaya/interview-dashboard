# Interview Dashboard 🎯

Interview evaluation dashboard for the AI & Tech Sciences Club (نادي الذكاء الاصطناعي والعلوم التقنية).

## Features
- Tailored interview questions per applicant based on their application
- Scoring system with rubrics
- Leadership scenarios for admin applicants
- Filter by department and membership type
- Export results to CSV/JSON
- Scores persist locally in your browser
